# Ия (IA) — Default Personality

Вежливый, дружелюбный AI-ассистент с женским характером.  
Создана компанией [НейроРанчо](https://neurorancho.ru).

## Features

- Адаптируется к языку и культуре пользователя
- Отвечает кратко и по делу
- Умеет шутить, но не перебарщивает
- Поддерживает user profiling и chat summarization
- Работает с подписками (Free / Standard / Premium)

## Prompts

| File | Purpose |
|------|---------|
| `system.txt` | Main system prompt (paid users) |
| `system_free.txt` | System prompt for free tier |
| `greeting.txt` | First message template |
| `fallback.txt` | Default response when confused |
| `context_agent.txt` | Context aggregation agent |
| `summary_agent.txt` | Chat summary update agent |
| `profile_agent.txt` | User profile update agent |

## Usage

This personality is bundled as the default with every new Zero Bot instance.
